Lagometer

class Lagometer(val frameDurationNs: Long, historyDuration: Duration = 15.seconds, historyResolution: Duration = 5.seconds)

Encapsulates the logic for measuring lag (drift) against a expected frame rate.

This class tracks the "drift" of incoming events (like game data packets) relative to the expected arrival time based on a constant frame rate. It accounts for a "leeway" buffer that allows for some jitter without counting as lag, which scales with the user's frame delay setting.
