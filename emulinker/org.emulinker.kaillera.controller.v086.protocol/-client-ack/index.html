ClientAck

data class ClientAck(val messageNumber: Int) : Ack, ClientMessage

This is effectively a ping response sent by the client when it receives a ServerAck.

This back and forth is used to establish a client's "ping" (millisecond delay) at login and to confirm that the client is still listening on the port.

Message type ID: 0x06.
